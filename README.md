## Этот проект создан только для демонстрации
Верстка **HTML5** с использованием фреймворка **Bootstrap** и **jQuery**
    Ссылки по сайту никуда не ведут и на нажатие демонстрируют описанные ниже эффекты.

### У анонсов статей:
- ссылка "*подробнее*" раскрывает по принципу аккордеон
- "*подробнее*" меняется на "*свернуть*"
- пример трансляции в соц.сети

### Блок обсуждений и форма ввода
- возможность `удаления сообщений`
- форма ввода

### Рекомендация!
Для разработки стилей, все работы проводятся **только** в файлах **LESS**.
Финально на странице подключается скомпилированный и минифицированный файл **css**.

### Файловая структура

имя файла  | cодержание
----------------|----------------------
/style/luzhok.css       | финальный файл стилей
/less/luzhok.less      | рабочий файл стилей
/less/mixins.less   | файл с примесями
/less/variables.less       | файл с переменными
/images/    | изображения по дизайну
/pics/ | текущие изображения (иконки/баннеры)
